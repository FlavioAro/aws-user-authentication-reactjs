# Getting Started

npx create-react-app aws-user-authentication-reactjs

npm install -g @aws-amplify/cli

amplify configure

server-amplify

amplify init

authreactapp

amplify add auth

amplify push

npm install --save aws-amplify @aws-amplify/ui-react
